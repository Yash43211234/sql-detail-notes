## 🔷 8. Excel vs SQL

### 📊 Excel
- Easy to use
- Good for small data
- Limitations:
  - Max ~1M rows
  - Slows for large data
  - Hard for complex logic

### 🗄️ SQL
- Handles large data efficiently
- Faster processing
- Supports complex queries

👉 Conclusion:
- Excel → Small data  
- SQL → Large & real-world data

---

## 🔷 9. What is SQL?
- SQL = Structured Query Language
- SQL does NOT store data

👉 Used to communicate with database

## 🔷 14. MySQL Installation Steps

### Step 1: Download
- Go to: mysql.com/downloads
- Select:
  - MySQL Community Server
  - LTS version

---

### Step 2: Install
- Follow setup steps
- Set:
  - Username: root
  - Password: your choice

---

### Step 3: Set Path
```bash
export PATH=$PATH:/usr/local/mysql/bin
```

---

### Step 4: Connect MySQL
```bash
mysql -u root -p
```

---

### Step 5: Check Databases
```sql
show databases;
```
